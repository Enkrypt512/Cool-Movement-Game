# Cool-Movement-Game
A WIP FPS movement game inspired by ULTRAKILL,CS:GO And Half-Life, made in Godot! With:​

- Settings

- Rebindable Keys

- Sprinting

- Crouching

- Sliding

- Freelooking

- Guns

- CS:GO-Like Recoil

- Controller Support

And More Coming Soon! Like:

- Decals

- Peeking

- Multiplayer

- Reloading Animation

- Stages

- Ledge Grabbing

- Hooks

- Melee Weapons
